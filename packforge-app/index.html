🛍 Etsy Integration
● Connect via Etsy Open API v3

Sync pack-ready counts, tier prices, and standalone item stock with your Etsy shop.

  1. Visit etsy.com/developers and sign in as a seller.
  2. Click "Create a New App" — name it "Pack Forge", redirect to http://localhost.
  3. Copy your API Key String from the app dashboard.
  4. Paste it below and click Connect to begin OAuth 2.0.
  5. Use Sync to Etsy buttons on each tier / standalone card to push qty and price.
↓ FROM ETSY
• Orders → decrement components
• Cancellations → restore stock
• Current listing quantities
↑ TO ETSY
• Packs-ready → listing qty
• Tier price → listing price
• Solo item stock → listing qty
• Sold-out auto-flag at 0
